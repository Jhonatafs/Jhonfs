Sintaxe dos dados Primitivos em Scheme

1. Comentários

Comentários são usados para anotar detalhes importantes no código. Em Scheme, comentários começam com ;; e continuam até o final da linha.


;; Este é um comentário
;; Outro comentário

2. Variáveis

Variáveis são usadas para armazenar e nomear dados. Em Scheme, você define uma variável usando o define.


;; Definindo uma variável
(define nome "Fulano")
(define idade 20)

;; Usando as variáveis
nome  ;; Retorna "Fulano"
idade ;; Retorna 20

3. Dados Primitivos

Os dados primitivos são os componentes básicos da linguagem. Em Scheme, os principais tipos de dados primitivos incluem:

3.1. Números

Números podem ser inteiros ou reais. Scheme suporta operações aritméticas básicas.


;; Números inteiros
(define a 10)
(define b 5)

;; Operações aritméticas
(+ a b)  ;; Soma: 15
(- a b)  ;; Subtração: 5
(* a b)  ;; Multiplicação: 50
(/ a b)  ;; Divisão: 2

3.2. Strings

Strings são sequências de caracteres. Você pode concatenar strings e acessar caracteres individuais.


;; Definindo strings
(define nome "Fulano")
(define sobrenome "de Tal")

;; Concatenando strings
(string-append nome " " sobrenome)  ;; Retorna "Fulano de Tal"

;; Acessando caracteres
(string-ref nome 0)  ;; Retorna #\F

3.3. Booleanos

Booleanos representam valores verdadeiros (#t) ou falsos (#f). Eles são usados em expressões condicionais.


;; Definindo booleanos
(define verdadeiro #t)
(define falso #f)

;; Expressões condicionais
(if verdadeiro "Verdadeiro" "Falso")  ;; Retorna "Verdadeiro"
(if falso "Verdadeiro" "Falso")        ;; Retorna "Falso"

3.4. Simbolos

Simbolos ou constantes são usados para representar nomes ou identificadores. Eles são imutáveis e único.


;; Definindo simbolos
(define id '0001)

;; Comparando simbolos
(eq? id '0001) ;; Retorna #t (Verdadeiro)
(eq? id '0002) ;; Retorna #f (Falso)

4. Funções

Funções são procedimentos que realizam tarefas específicas. Em Scheme, você define uma função usando define e lambda.


;; Definindo uma função simples
(define (quadrado x)
  (* x x))

;; Usando a função
(quadrado 5)  ;; Retorna 25

4.1. Funções Anônimas (Lambda)

Uma função lambda é uma função anônima, ou seja, uma função que não tem nome. Em Scheme, funções lambda são criadas usando a palavra-chave lambda. Elas são úteis para criar funções temporárias ou para passar funções como argumentos para outras funções.


;; Definindo uma função anônima
((lambda (x) (* x x)) 3)  ;; Retorna 9

;; Usando lambda em uma função definida
(define quadrado (lambda (x) (* x x)))
(quadrado 3)  ;; Retorna 9
